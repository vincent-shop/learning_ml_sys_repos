# PyTorch (submodule)

This folder vendors **PyTorch** as a git submodule:

- **Upstream**: `https://github.com/pytorch/pytorch`
- **Local path**: `examples/pytorch/pytorch`

## What it is

PyTorch is a deep learning framework providing:

- Tensor computation (NumPy-like) with strong GPU acceleration
- Tape-based autograd + `torch.nn` for building neural networks

- `pytorch/README.md` — overview + install (binaries / from-source) + resources
- `pytorch/CONTRIBUTING.md` — codebase structure + dev workflow, linting, and testing

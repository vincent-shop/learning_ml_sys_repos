# ml-systems-papers (submodule)

This folder vendors **ml-systems-papers** as a git submodule:

- **Upstream**: `https://github.com/byungsoo-oh/ml-systems-papers`
- **Local path**: `examples/ml-systems-papers/ml-systems-papers`

## What it is

A curated, topic-organized paper list for **Machine Learning Systems** (surveys are
tagged with a “Survey” prefix).

## Start here (inside the submodule)

- `ml-systems-papers/README.md` — the full list + table of contents

## Topic coverage (from upstream TOC)

- Data processing (pipelines, caching, LLM data plane)
- Training systems (scheduling, distributed training, etc.)
- Inference systems, attention optimization, MoE, networking/communication, FT/stragglers,
  GPU memory/sharing, compilers/kernels, long context, compression, RAG, agentic systems,
  RL post-training, multimodal, and more.

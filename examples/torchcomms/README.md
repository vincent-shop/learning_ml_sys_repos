# torchcomms (submodule)

This folder vendors **torchcomms** as a git submodule:

- **Upstream**: `https://github.com/meta-pytorch/torchcomms`
- **Local path**: `examples/torchcomms/torchcomms`

## What it is

An experimental communications API for PyTorch, providing a high-level collectives
API and multiple backends (NCCL/NCCLX, Gloo, RCCL/RCCLX, etc.).

## Start here (inside the submodule)

- `torchcomms/README.md` — prerequisites + install (pip nightly / from source) + quick start example
- `torchcomms/comms/torchcomms/examples/` — example code
